# Challenge Submission Notes

## Project title

International Football Match Predictor

## Short description

A Python/Jupyter machine learning proof of concept that predicts international football match outcomes using historical match results and an interactive Streamlit prototype.

## IBM AI-supported technology

IBM Bob. The project follows the IBM SkillsBuild football prediction lab workflow and documents Bob prompts used for code generation, explanation, debugging, and prototype refinement.

## Repository checklist

- Public GitHub repo
- `README.md` with project background, data source, model method, run instructions, IBM Bob usage, limitations, and future improvements
- `data/results.csv` from the official lab dataset
- `notebooks/corelab.ipynb` completed project walkthrough
- `app.py` Streamlit prototype
- `src/` training and prediction code
- IBM Bob usage screenshots in `docs/images/`
- No secrets, tokens, private emails, or local machine paths in committed text

## Demo video checklist

- Maximum length: 3 minutes
- Show repo and README
- Show notebook or training command
- Run Streamlit app
- Predict at least one matchup, for example Brazil vs Argentina
- Show the IBM Bob evidence section in the README
- Mention that Bob suggestions were verified before use
- Mention limitations
